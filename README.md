## Wathe Train Utils 🚂
A Minecraft Datapack addon for the Wathe mod by Youtuber Doctor4t. Adds a few utilities that help make sessions easier to manage/handle

# Features:
 - Adds 2 custom goat horns:
   > Conductor's Horn - Queues a Game-Start on use, 20s delay with use
   > Beckoner's Horn - Teleports all players to you on use, single-use
 - Adds Conductors - players who have the Conductor's Horn, they have 
   the ability to start games
 - Adds configurability with scoreboards:
   > Time duration of sessions, 
   > Modded/Vanilla Session compatibility (slightly buggy as of now), 
   > Maximum number of Conductors

And More planned eventually!
